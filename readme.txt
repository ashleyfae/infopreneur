== Installation ==

1. Download the latest zip file from the [release page](https://github.com/nosegraze/infopreneur/releases).
1. Unzip the archive and rename the folder to `infopreneur`. Then re-zip the file.
1. Login to WordPress, go to Appearance -> Themes, click "Add New", then click "Upload Theme".
1. Upload the zip file.
1. Activate the theme.
1. Change the settings in Appearance -> Customize.
